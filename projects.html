PDU Monitoring System REST API
About

PDU monitoring system is a web-based system that refers to PT PDU's previous monitoring system, DOME. DOME (Drilling Online Monitoring Environment) is a system that provides accurate real-time monitoring of the drilling process, allowing users to make timely decisions based on the data provided. The PDU Monitoring System is a redesign of DOME. In addition, the PDU Monitoring System also adds a machine learning-based automatic notification feature to detect potential stuck drill situations. PDU Monitoring System REST API is a backend REST API for the PDU Monitoring System that provides several endpoints, such as register, login, employee, company, place, well, and records.
